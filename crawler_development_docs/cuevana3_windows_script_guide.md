# Cuevana3 爬虫系统 - Windows 一键脚本使用指南

## 🚀 快速开始

### 1. 安装 Python

如果您还没有安装 Python，请按以下步骤操作：

1. 访问 Python 官网：https://www.python.org/downloads/
2. 下载 Python 3.7 或更高版本
3. 安装时**务必勾选** "Add Python to PATH"
4. 完成安装后，打开命令提示符，输入 `python --version` 验证安装

### 2. 运行一键脚本

**双击运行：**
```
cuevana3_launcher.bat
```

或者在命令提示符中：
```cmd
cd C:\path\to\cuevana3_v2
cuevana3_launcher.bat
```

### 3. 首次运行

首次运行时，脚本会自动：
- ✅ 检查 Python 是否安装
- ✅ 自动安装所需依赖包
- ✅ 创建默认配置文件

## 📋 主菜单功能

### 【电影爬取】

#### 1. 爬取单个年份的电影
- 输入年份（例如：2025）
- 可选：输入最大页数限制
- 系统会自动爬取该年份的所有电影

**示例：**
```
请输入年份: 2025
请输入最大页数限制 (留空表示不限制): 5
```

#### 2. 爬取年份范围的电影（倒序）
- 输入起始年份（例如：2020）
- 输入结束年份（例如：2025）
- 系统会从最新年份开始倒序爬取

**示例：**
```
请输入起始年份: 2020
请输入结束年份: 2025
```
爬取顺序：2025 → 2024 → 2023 → 2022 → 2021 → 2020

#### 3. 快速爬取最近2年电影
- 一键操作，无需输入
- 自动爬取最近2年的电影（每年最多10页）

### 【电视剧爬取】

#### 4. 爬取电视剧（自定义数量）
- 输入要爬取的电视剧数量
- 可选：输入列表最大页数

**示例：**
```
请输入要爬取的电视剧数量: 50
请输入电视剧列表最大页数: 3
```

#### 5. 快速爬取20部电视剧
- 一键操作，无需输入
- 自动爬取最近20部电视剧的所有剧集

### 【数据管理】

#### 6. 更新所有数据（推荐定时任务）
- 自动更新最近2年的电影
- 自动更新最近20部电视剧
- **推荐用于定时任务**

#### 7. 导出数据到 CSV
选择导出类型：
- **1** - 仅导出电影
- **2** - 仅导出电视剧
- **3** - 导出所有数据

导出的文件会保存到配置的导出目录中。

#### 8. 查看数据库统计
显示：
- 电影总数、年份数、平均评分
- 电视剧总数、独立剧集数、平均评分
- 最新添加的内容

#### 9. 清空数据库
- ⚠️ **危险操作**：会删除所有数据
- 需要输入 `YES` 确认

### 【系统设置】

#### A. 配置管理
进入配置管理菜单，可以设置：

**1. 设置数据库路径**
- 默认：`cuevana3.db`（当前目录）
- 可以修改为任意路径，例如：
  - `D:\Data\cuevana3.db`
  - `C:\Users\YourName\Documents\cuevana3.db`

**2. 设置导出目录**
- 默认：`.`（当前目录）
- 可以修改为任意目录，例如：
  - `D:\Exports`
  - `C:\Users\YourName\Downloads`

**3. 设置延迟时间**
- 默认：1.0-3.0秒
- 可以调整以控制爬取速度
- 建议不要低于0.5秒

**4. 重置为默认配置**
- 恢复所有设置为默认值

#### B. 查看日志
- 显示最近的日志文件
- 可以打开日志目录查看详细日志

#### C. 打开项目目录
- 快速打开项目所在文件夹

## ⚙️ 配置文件说明

配置文件：`config.ini`

### 默认配置

```ini
[Database]
database_path = cuevana3.db

[Export]
export_directory = .
movies_filename = movies.csv
tv_series_filename = tv_series.csv

[Scraper]
delay_min = 1.0
delay_max = 3.0
max_retries = 3
```

### 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `database_path` | 数据库文件路径 | `cuevana3.db` |
| `export_directory` | 导出文件保存目录 | `.` (当前目录) |
| `movies_filename` | 电影CSV文件名 | `movies.csv` |
| `tv_series_filename` | 电视剧CSV文件名 | `tv_series.csv` |
| `delay_min` | 最小延迟时间（秒） | `1.0` |
| `delay_max` | 最大延迟时间（秒） | `3.0` |
| `max_retries` | 最大重试次数 | `3` |

## 💡 使用技巧

### 1. 自定义保存路径

**场景：我想把数据保存到 D 盘**

1. 运行脚本，选择 `A` 进入配置管理
2. 选择 `1` 设置数据库路径
3. 输入：`D:\cuevana3_data\cuevana3.db`
4. 选择 `2` 设置导出目录
5. 输入：`D:\cuevana3_data\exports`

### 2. 快速更新数据

**场景：我想每天自动更新最新数据**

- 方法1：每天手动运行脚本，选择 `6` 更新所有数据
- 方法2：使用 Windows 任务计划程序自动执行（见下文）

### 3. 批量爬取历史数据

**场景：我想爬取2015-2025年的所有电影**

1. 选择 `2` 爬取年份范围
2. 起始年份：`2015`
3. 结束年份：`2025`
4. 每年最大页数：留空（不限制）

### 4. 导出到指定位置

**场景：我想把CSV文件导出到桌面**

1. 选择 `A` 进入配置管理
2. 选择 `2` 设置导出目录
3. 输入：`C:\Users\YourName\Desktop\cuevana3_exports`
4. 返回主菜单，选择 `7` 导出数据

## ⏰ 设置 Windows 定时任务

### 方法1：使用任务计划程序（推荐）

1. **打开任务计划程序**
   - 按 `Win + R`
   - 输入 `taskschd.msc`
   - 按回车

2. **创建基本任务**
   - 点击右侧 "创建基本任务"
   - 名称：`Cuevana3 自动更新`
   - 描述：`每天自动更新电影和电视剧数据`

3. **设置触发器**
   - 选择 "每天"
   - 开始时间：`凌晨 2:00`

4. **设置操作**
   - 选择 "启动程序"
   - 程序或脚本：`python.exe`
   - 添加参数：`main.py update`
   - 起始于：`C:\path\to\cuevana3_v2`

5. **完成设置**
   - 勾选 "完成时打开此任务属性的对话框"
   - 点击 "完成"

6. **高级设置**（可选）
   - 在属性对话框中，切换到 "条件" 选项卡
   - 取消勾选 "只有在计算机使用交流电源时才启动此任务"
   - 切换到 "设置" 选项卡
   - 勾选 "如果任务失败，每隔以下时间重新启动"

### 方法2：使用批处理脚本

创建一个新的批处理文件 `auto_update.bat`：

```batch
@echo off
cd /d C:\path\to\cuevana3_v2
python main.py update
pause
```

然后在任务计划程序中直接运行这个批处理文件。

## 🔍 常见问题

### Q1: 双击脚本后闪退？

**A:** 可能是 Python 未正确安装或未添加到 PATH。

**解决方案：**
1. 重新安装 Python，确保勾选 "Add Python to PATH"
2. 或者右键点击脚本，选择 "编辑"，在开头添加：
   ```batch
   pause
   ```
   这样可以看到错误信息

### Q2: 提示缺少依赖包？

**A:** 脚本会自动安装依赖，如果失败，请手动安装：

```cmd
cd C:\path\to\cuevana3_v2
pip install -r requirements.txt
```

### Q3: 爬取速度太慢？

**A:** 可以调整延迟时间：

1. 选择 `A` 进入配置管理
2. 选择 `3` 设置延迟时间
3. 输入更小的值，例如：`0.5` 和 `1.5`

⚠️ 注意：延迟时间太短可能触发反爬虫机制

### Q4: 数据保存在哪里？

**A:** 默认保存在脚本所在目录：

- 数据库：`cuevana3.db`
- 导出的CSV：当前目录下的 `movies.csv` 和 `tv_series.csv`

可以通过配置管理修改保存路径。

### Q5: 如何查看爬取日志？

**A:** 
1. 选择 `B` 查看日志
2. 或者直接打开 `logs` 文件夹
3. 日志文件名格式：`scraper_YYYYMMDD_HHMMSS.log`

### Q6: 如何备份数据？

**A:** 只需复制以下文件：
- `cuevana3.db` - 数据库文件
- `config.ini` - 配置文件（可选）

### Q7: 中文显示乱码？

**A:** 这是 Windows 命令提示符的编码问题。

**解决方案：**
1. 使用 Windows Terminal（推荐）
2. 或者在命令提示符中执行：`chcp 65001`

### Q8: 如何清空所有数据重新开始？

**A:** 
1. 选择 `9` 清空数据库
2. 输入 `YES` 确认
3. 或者直接删除 `cuevana3.db` 文件
